1- git pull / git clone (to pull project localy)
2- composer install (to install vendort and all laravel dependencies)
3- cp .env.example .env (to create .env file)
4- php artisan key:generate (to create the laravl key)
5- php artisan jwt:secret (to create th jwt key for authentification)
6- php artisan migrate (after running mysql to create databases)
7- php artsian migrate:fresh --seed (to use seeded data in testing or "php artisan db:seed" if u have data that you don't want to get droped)
